## Response 解析

已经理解并实际观察：

```python
response.choices[0].message.content
```

并分析过以下字段：

- `model`
- `choices`
- `message`
- `content`
- `finish_reason`
- `usage`
- `prompt_tokens`
- `completion_tokens`
- `total_tokens`
- `reasoning_tokens`
- `provider`
- `tool_calls`
- `cost`

已理解：

- `choices[0]` 表示第一个候选结果；
- `finish_reason="stop"` 表示正常结束；
- `finish_reason="length"` 可能表示回答被截断；
- Prompt Token 不仅包含用户输入，还包括 System Prompt 和历史消息；
- `openrouter/free` 是路由名称，实际模型可能是其他免费模型；
- 业务代码不应依赖非通用的 reasoning 字段。
